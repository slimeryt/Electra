Features

Everything you need,
nothing you don't.

Built from scratch with performance and privacy in mind. No bloat, no tracking, no nonsense.

Real-time Messaging
Instant messages via WebSockets. Servers, channels, direct messages, typing indicators — everything works instantly.
Voice & Video
Crystal-clear P2P voice and video powered by WebRTC. Join voice channels, share your camera or screen with one click.
File Sharing
Drag & drop files directly into chat. Images show inline previews. Supports any file type up to 50 MB.
Secure Auth
JWT access tokens with short expiry, bcrypt-hashed passwords, and rotating refresh tokens. Your data stays yours.
Desktop App
Native Windows app built with Electron. System tray, auto-updater, frameless window, deep links — it feels like a real app.
Self-hostable
Open source Node.js backend with SQLite. Run it on your own server for complete control. No subscription required.
Get started

Up and running
in minutes.

No configuration needed. Download, install, and you're in.

01
Download & Install
Grab ElectraInstaller.exe and run it. Installs to C:\Users\you\Electra — no admin needed.
02
Create an Account
Register with a username and password. No email verification, no phone number, no nonsense.
03
Create or Join a Server
Start your own server in one click, or join an existing one with an invite code. Invite your friends and start chatting.
