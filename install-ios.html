Use Electra on iPhone & iPad

Apple does not allow installing arbitrary apps outside the App Store the same way Android does. You can still run Electra like a native app by adding the web app to your Home Screen from Safari.

Which URL should I use?

Use the address of the Electra web app — the site that shows the real app: Login / Register, then servers and channels (purple UI). That is usually not the same link as this marketing page or a “download only” site.

  • Correct: an https:// link where you can sign in and use chat like on a PC browser.
  • Not the API URL: developers configure something like https://…railway.app for the backend only. You do not paste that into Safari unless your host also serves the app there.
  • Not this brochure site: pages that only describe Electra or offer Windows/Android downloads are not the app — adding those to Home Screen will not give you chat.
Quick check: open the link in Safari first. If you see username/password (or register) and the app layout, use that address for Add to Home Screen.
Don’t have a web app link? Whoever runs your server must deploy the built frontend (npm run build in the Electra frontend) to HTTPS — for example Railway, Vercel, Netlify, or nginx — and share that public URL with you.
HTTPS required: iOS needs a secure (https://) address. If you only have http://, Safari may block features or the shortcut may behave poorly.
  1. Open Safari

    On iOS you must use Safari (not Chrome or in-app browsers) for the best “Add to Home Screen” experience.

  2. Open your web app URL

    Type or paste the Electra app address (the one that shows login — see Which URL? above). Sign in and wait until the main app has fully loaded.

  3. Open the Share menu

    Tap the Share button (square with an arrow pointing up) at the bottom on iPhone, or top on iPad.

  4. Add to Home Screen

    Scroll down and tap Add to Home Screen. Choose a name (e.g. Electra) and tap Add.

  5. Launch from the icon

    Open Electra from your new home screen icon. It opens full screen, similar to a standalone app. Microphone and camera may prompt for permission when you join voice or video.

Tip: keep Electra open in Safari once before adding to Home Screen so icons and offline cache can load. For updates, pull to refresh inside the app or clear Safari cache if something looks stuck.
